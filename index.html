🧪 Sobre este Laboratorio

Este prototipo de aprendizaje inmersivo combina visualización 3D interactiva con inteligencia artificial para enseñar conceptos fundamentales de Química I.

Características:

  • Visualización 3D: Manipula moléculas y observa enlaces en tiempo real
  • Tutor IA: Responde preguntas y guía tu aprendizaje
  • Accesibilidad: Diseñado siguiendo principios DUA (Diseño Universal para el Aprendizaje)
  • Multiplataforma: Funciona en navegadores modernos sin instalación

Tecnologías:

  • Spline para gráficos 3D
  • JavaScript ES6+ puro
  • Tailwind CSS
  • Node.js + Express (backend opcional)

⚠️ Nota: Este es un prototipo educativo. Para usar el tutor IA con OpenAI, configura el backend siguiendo las instrucciones del README.
